# Manual-Testing---SauceDemo-Swag-Labs-
Manual testing project performed on SauceDemo (Swag Labs) covering test case design, execution, and defect reporting.
## Objectives
- Design test cases based on application requirements
- Execute test cases and record results
- Identify and report defects

##  Scope of Testing
- Login Functionality
- Product Listing
- Add to Cart
- Checkout Process
- Logout Functionality

## Testing Type
- Functional Testing
- UI Testing
- Regression Testing (basic)

## Test Artifacts
- Test Cases: `testcases.xlsx`
- Test Execution Report: Included in test cases

##  How to Use
1. Open the test cases Excel file
2. Review test scenarios and steps
3. Check execution results and status

## Application Under Test
https://www.saucedemo.com/
